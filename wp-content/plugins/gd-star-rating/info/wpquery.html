wp query

Query variables

  • gdsr_sort: data used for sorting results
  • gdsr_order: data sorting order
  • gdsr_multi: standard or multi ratings data to use
  • gdsr_fsvmin: filter to specify minimal number of standard rating votes
  • gdsr_ftvmin: filter to specify minimal number of thumbs rating votes

gdsr_sort

Supported values (columns) for sorting data:

  • rating: post ratings, standard or multi ratings depending on the multi parameter
  • review: post review rating, standard or multi ratings depending on the multi parameter
  • votes: number of votes saved for the post
  • thumbs: post thumbs rating
  • last_voted: date of the last vote recorded

gdsr_order

Ascending or descending order to sort data:

  • asc: ascending
  • desc: descending (default)

gdsr_multi

For standard rating data, don't use this variable, or set it to zero.
To get multi ratings data this variable must be set to multi set id to use.

gdsr_fsvmin

Set to number of votes for standard rating, and only posts with at least set number of votes will be included.

gdsr_ftvmin

Set to number of votes for thumbs rating, and only posts with at least set number of votes will be included.
