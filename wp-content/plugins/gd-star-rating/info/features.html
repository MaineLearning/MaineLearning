GD Star Rating

plugin features

Rating & Review

  • Rating of posts and pages
  • Rating of comments
  • Thumbs up/down rating of posts and pages
  • Thumbs up/down rating of comments
  • Multiple ratings for posts and pages
  • Review rating for posts and pages
  • Review rating for posts and pages with multi ratings
  • Review rating for comments

Unique Features

  • Data prefetch and caching
  • Internal CSS optimization
  • Builder panel for getting shortcodes and functions

Integration

  • Modifing WP query to sort posts by ratings data
  • Widget with list of ratings
  • Widget with top blog rating
  • Widget with post comments ratings
  • Widgets with multi instance support
  • Widgets support templates
  • WordPress shortcode support
  • WordPress editor button and tinyMCE3 plugin for shortcode
  • RSS Integration for powered by button
  • RSS Integration for post ratings
  • Expanding plugin using WordPress actions and filters
  • Manual placement functions of the rating code and template integration
  • Support for QTranslate plugin

Rendering & Graphics

  • Advanced T2 templates system
  • Charting support with pChart library
  • Administration dasboard widget
  • Alternative rendering to generate rating stars as static images
  • Preview stars, thumbs and trends image sets
  • Different trend, thumbs and star styles and sizes
  • Select sets to include in main CSS file
  • Cache support for storing rendered images

Voting / Rating

  • Voting rules on post/page base
  • Voting rules based on categories
  • Auto inserting rating code
  • Moderation of votes
  • Saving of voting data for registered users and regular visitors
  • Saving IP and user agent data for each vote
  • Calculation of voting trend data for each post, page and comment

Others

  • Support for cache plugins
  • Ajax rating support using jQuery
  • Multi language support
  • Post and comments based votes administration
  • User based votes administrations
  • Saving cookies with votes
  • Various additional tools for maintenace and bulk operations
  • Banning IP's, IP ranges and masked IP's
  • Filtering out search bots
  • Export data into csv file
  • Extensive PDF User Guide

Import Data

Plugin can import data from different rating plugins:

  • WP Post Rating http://wordpress.org/extend/plugins/wp-postratings/
  • Star Rating For Reviews http://wordpress.org/extend/plugins/star-rating-for-reviews/
  • Post Star Rating http://wordpress.org/extend/plugins/post-star-rating/
